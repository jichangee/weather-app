# 7 天天气走势图设计

## 目标

将现有的 7 天天气列表改为走势图，在同一张卡片里同时展示最高温和最低温。最高温信息位于上半区，最低温信息位于下半区，天气图标放在两条温度走势之间偏上的位置。

## 方案

新增一个 `DailyTrendChart` 组件，输入使用现有 `DailyItem[]`。组件用 SVG 绘制 7 天温度走势：

- 顶部展示每天日期标签。
- 上半区绘制最高温折线，并在点位上方显示最高温数值。
- 中间展示每天的天气图标。
- 下半区绘制最低温折线，并在点位下方显示最低温数值。
- 底部展示天气文字，作为图标的补充信息。

## 组件边界

`DailyTrendChart` 只负责渲染 7 天天气趋势，不负责请求数据。`Home` 继续从 `weatherDaily.daily` 读取数据，并在 7 天天气卡片中替换现有列表。

## 视觉约束

走势图沿用当前玻璃卡片、白色半透明文字和蓝色趋势线风格。图表使用固定 SVG `viewBox` 和响应式宽度，避免不同屏幕下布局跳动。温度曲线根据 7 天最高/最低温共同计算比例，确保两条曲线可比较。

## 测试

使用现有构建链路验证 TypeScript 和 Next.js 构建通过。重点检查空数据或少于两天数据时组件不渲染，避免 SVG 计算异常。
